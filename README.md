# Password Strength Analyzer
Analisador moderno de força de senhas.



## Funcionalidades

<ul>
  <li>Análise de senha em tempo real</li>
  <li>Barra de força</li>
  <li>Estimativa de tempo de quebra</li>
  <li>Simulação de brute force</li>
  <li>Recomendações de segurança</li>
  <li>Verificação de vazamentos</li>
  <li>Interface responsiva</li>
</ul>



## Tecnologias

<ul>
  <li>Next.js</li>
  <li>TypeScript</li>
  <li>Tailwind CSS</li>
  <li>Framer Motion</li>
  <li>zxcvbn</li>
  <li>CryptoJS</li>
</ul>
